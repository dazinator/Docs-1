Building Modules

Overview

A DNN module is custom code that extends the functionality of the DNN platform. For example, a module:

  • can generate dynamic content, which is displayed in a pane of a webpage, or
    • or display content from another source
  • edit content . or manage data or change database configuration or provide administrative functions. or provide other functionality.

Example: If you want a pane to display a stock ticker or a weather forecast, you can build a module to do so.
