Introduction to DNN Modules

Overview

A DNN module is custom code that extends the functionality of the DNN platform. A module can:

  • generate dynamic content,
  • display content from another source,
  • edit content,
  • manage data,
  • change database configuration,
  • provide administrative functions, or
  • provide other functionality.

Tip: Before you try to build your own module, check if one already exists with the functionality you want. Hundreds of third-party modules are available from these sources:

  • For sale: DNN Store - Some developers include their source code for a slightly higher price.
  • Open source and free:
    • DNN Forge
    • DNN at Codeplex
    • DNN at GitHub

The DNN Platform provides certain functionality by default, including security settings.
