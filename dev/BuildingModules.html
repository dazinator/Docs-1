Building Modules

Overview

You can build an entire module from scratch; however, the easiest way is to customize a set of module development templates.

Before you try to build your own module, check if one already exists with the functionality you want. Hundreds of third-party modules are available from these sources:

  • For sale: DNN Store - Some developers provide versions that include the source code.
  • Open source and free:
    • DNN Forge
    • DNN at Codeplex
    • DNN at GitHub
