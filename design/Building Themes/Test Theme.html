Test the Theme on a Web Page

Prerequisites

Steps

  1. Create a new web page.
  2. Edit the page settings and go to Advanced Settings.
  3. In the Page Theme section, select Host and choose one of the layout templates from your new theme.
    Note: When a layout template is added to the page, the associated CSS file is automatically added.
  4. Click Update.
  5. View your page in different browsers to verify that the new theme was applied correctly.
