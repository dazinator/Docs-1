Create a CSS File

Overview

CSS files can be associated with a layout template or container by giving it the same filename and placing it in the same folder. When a layout template (or container) is applied to a webpage (or pane), the associated CSS file is automatically applied. Example: These three files are automatically associated with each other:

  • MyLayout.ascx
  • MyLayout.css
  • MyLayout.doctype.xml

CSS files are applied hierarchically.

  1. If a layout template or container does not have a specific CSS associated with it, the master CSS for the theme is used. The master CSS for the theme must be called skin.css.
  2. If the theme does not have a master CSS, then the default theme for the website is used.
  3. If the website does not have a master CSS, then the default theme for the host is used.

Steps

  1. For every pane (<div>) that contains a theme object, add a style definition in the CSS file. The style definition must start with # followed by the div id.
    Example:
    layout templateCSS
    <div id="login_style" class="user"> <object id="dnnLOGIN" codebase="LOGIN" codetype="dotnetnuke/server"> <param name="CssClass" value="user" /> </object> </div> #login_style .linkseparator{ color: white; font-weight: bold; }
