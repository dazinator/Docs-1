Convert the Layout Template from HTML to ASCX

Prerequisites

Steps

  1. While logged in as an admin or host, go to Admin > (double-gear tab) > Themes.
  2. Choose your new theme in the Themes dropdown, then click Parse Theme Package.

Example:

This HTML layout template is converted to this ASCX layout template
<div id="wrapper"> <div id="ContentPane" runat="server"></div> </div> <%@ Control language="vb" AutoEventWireup="false" Explicit="True" Inherits="DotNetNuke.UI.Skins.Skin"> <div id="ContentPane" runat="server"></div>

NOTE: The conversion only goes in one direction (from HTML to ASCX). To make changes to the layout template, you can:

  • make changes directly to the ASPX file, or
  • make the changes in your original HTML file and reconvert it to ASPX, overriding any changes made directly to the ASPX file.
