Convert the Layout Template from HTML to ASCX

Prerequisites

  • A super user (host) account. Super user privileges are required to upload themes, because themes can include executables. The super user also converts the theme from HTML to ASCX. (An administrator account is sufficient to apply themes and containers to a site.)

Steps

  1. In the DNN administration page, go to Admin > (double-gear tab) > Themes.
  2. Choose your new theme in the Themes dropdown, then click Parse Theme Package.

RESULT: DNN converts the very basic layout template (index.html) to a new file (index.ascx) similar to this :

<%@ Control language="vb" AutoEventWireup="false" Explicit="True" Inherits="DotNetNuke.UI.Skins.Skin" > <div id="ContentPane" runat="server" ></div>

NOTE: The conversion only goes in one direction (from HTML to ASCX).
