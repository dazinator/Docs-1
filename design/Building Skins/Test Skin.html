Test the Skin on a Web Page

Prerequisites

  • A super user (host) account. Super user privileges are required to upload skins, because skins can include executables. The super user also converts the skin from HTML to ASCX. (An administrator account is sufficient to apply skins and containers to a site.)

Steps

  1. Create a new web page.
  2. Edit the page settings and go to Advanced Settings.
  3. In the Page Skin section, select Host and choose one of the layout templates from your new skin.
    Note: When a layout template is added to the page, the associated CSS file is automatically added.
  4. Click Update.
  5. View your page in different browsers to verify that the new skin was applied correctly.
