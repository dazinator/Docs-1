Create a CSS File

Steps

  1. Create a CSS file in the same folder as the layout template.
  2. For every section (<div>) that contains a skin object in the layout template, add a style definition in the CSS file. The style definition must start with # followed by the div id.
    Example:
    layout templateCSS
    <div id="login_style" class="user"> <object id="dnnLOGIN" codebase="LOGIN" codetype="dotnetnuke/server"> <param name="CssClass" value="user" /> </object> </div> #login_style .linkseparator{ color: white; font-weight: bold; }
