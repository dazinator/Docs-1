Create a DocType XML

Prerequisites

  • A super user (host) account. Super user privileges are required to upload skins, because skins can include executables. The super user also converts the skin from HTML to ASCX. (An administrator account is sufficient to apply skins and containers to a site.)

Overview

If a page is not rendering properly in some browsers, you can force the skin to be rendered in XHTML 1.0 Transitional by creating a doctype XML file.

Steps

  1. Create a doctype XML file with the same name as your layout template file, followed by .doctype.xml.
    Example: If your layout template file is called MyLayout.html, give your doctype file the name MyLayout.doctype.xml.
  2. Enter the following in your doctype file: <SkinDocType> <![CDATA[<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">]]> </SkinDocType>
