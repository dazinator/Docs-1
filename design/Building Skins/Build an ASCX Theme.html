Build an ASCX Theme

Overview

The most basic DNN theme configuration is comprised of one layout template. However, you can also create one CSS file for each layout template or a master CSS file for all layout templates.

Prerequisites

  • A super user (host) account. Super user privileges are required to upload themes, because themes can include executables. (An administrator account is sufficient to apply themes and containers to a site.)

Steps

  1. Create a layout template for the theme in ASCX format.
  2. (Optional) Create a CSS file for each layout template, or create a single master CSS file to be used for all layout templates.
  3. Test the theme on a webpage.
  4. (Optional) Create a doctype XML file.
  5. Package your DNN theme.
