Uploading Themes to Your DNN Website

Prerequisites

Steps

  1. While logged in as a host, go to Host > File Management.
  2. In your DNN installation, create a new folder for your theme under Themes\_default. The new folder name becomes the name of the DNN theme.

  3. Upload these files to the new DNN theme folder:
    • layout templates for the site
    • CSS files for the site (These must be in the same folder as the associated layout templates.)
    • containers for modules
